# Skill: Company Card Layouts (biznesinfo)

Варианты размещения логотипа на странице компании.

## Текущий стандарт

**По умолчанию для всех компаний:** Вариант 2 (логотип слева)

## Доступные варианты

### Вариант 1: По центру над названием (`logoPosition: "top"`)
Логотип по центру в шапке, название под ним.
```
        [ЛОГО]
   Название компании
      Категория
```

### Вариант 2: Слева от названия (`logoPosition: "left"`) ⭐ СТАНДАРТ
Логотип слева, название и категория справа. Компактный вариант.
```
[ЛОГО]  Название компании
        Категория • Город
```

### Вариант 3: В блоке контактов (`logoPosition: "contacts"`)
Логотип в секции контактов рядом с адресом и телефонами.

### Вариант 4: Отдельным блоком (`logoPosition: "default"`)
Логотип отдельным блоком в контенте под шапкой.

---

## Как применить вариант к конкретной компании

### 1. Добавить компанию в особые
Файл: `app/src/lib/companyOverrides.ts`

```typescript
export const COMPANY_OVERRIDES: Record<string, CompanyOverride> = {
  "msu-23": {
    logoPosition: "left",  // вариант 2
  },
  "another-company": {
    logoPosition: "top",   // вариант 1
  },
};
```

### 2. Пересобрать и перезапустить
```bash
cd /home/mlweb/biznesinfo.lucheestiy.com
docker compose build app
docker compose up -d app
```
