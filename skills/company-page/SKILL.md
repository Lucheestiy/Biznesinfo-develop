## 🎨 Цветовая схема

### Основные цвета (градиент маджента)
| Название | HEX | Использование |
|----------|-----|---------------|
| Светлый | `#b10a78` | Начало градиента |
| Основной | `#820251` | Бренд-цвет, акценты |
| Тёмный | `#7a0150` | Конец градиента |

### Градиенты
```css
/* Шапка */
bg-gradient-to-br from-[#b10a78] via-[#820251] to-[#7a0150]

/* Меню */
bg-gradient-to-r from-[#820251] to-[#9a0660]
```

### Дополнительные
- Белый текст: `text-white`
- Полупрозрачный: `text-white/80`, `bg-white/20`
- Фон страницы: `bg-gray-100`
- Карточки: `bg-white rounded-lg shadow-sm`

## 🖼️ Шапка (Header)

### Стандартный макет (логотип слева)
```
┌──────────────────────────────────────┐
│  ┌────┐                              │
│  │ЛОГО│  Название компании           │
│  │    │  Категория → Рубрика         │
│  └────┘                              │
│                                      │
│  Краткое описание компании...        │
└──────────────────────────────────────┘
```

**Размеры логотипа:**
- Мобильный: 80×80px (`w-20 h-20`)
- Десктоп: 112×112px (`md:w-28 md:h-28`)

**Шрифты:**
- Название: `text-xl md:text-3xl lg:text-4xl font-bold`
- Категория: `text-sm md:text-base text-white/80`

### Декоративные элементы
- Полупрозрачные круги (`bg-white/5 rounded-full`)
- Hero-изображение с blur (если есть)

## 📍 Меню-якоря (Anchor Menu)

**Sticky** меню с горизонтальным скроллом.

| Раздел | Иконка | Якорь | Условие показа |
|--------|--------|-------|----------------|
| Контакты | 📞 | #contacts | Всегда |
| О компании | 📋 | #about | Всегда |
| Наши услуги | ⚡ | #services | Если есть услуги или рубрики |
| Фото | 🖼️ | #photos | Если есть фото |
| Отзывы | ⭐ | #reviews | Если есть отзывы |
| Ключевые слова | 🏷️ | #keywords | Всегда |
| Карта | 🗺️ | #map | Всегда |

**⚠️ Пустые разделы скрываются автоматически!**
- Нет услуг и рубрик → секция "Наши услуги" не показывается
- Нет фото → секция "Фотогалерея" не показывается  
- Нет отзывов → секция "Отзывы" не показывается

**Стиль кнопок:**
```css
bg-white/20 rounded-lg text-white 
hover:bg-white/30 border border-white/30
```

## 📞 Блок контактов

### Порядок отображения
1. **Адрес** — 📍 индекс, город, улица
2. **Телефоны** — 📞 с подписями (приёмная, снабжение, бухгалтерия)
3. **Email** — ✉️ ссылка на почту
4. **Сайт** — 🌐 только домен
5. **Соцсети** — отдельными карточками с иконками
6. **Режим работы** — часы, перерыв

### Кнопки действий
- **Позвонить** — зелёная `bg-green-600`
- **Написать** — обводка `border-[#820251]`
- **AI-ассистент** — опционально

## 🔧 Персонализация

### Файл переопределений
`app/src/lib/companyOverrides.ts`

```typescript
export const COMPANY_OVERRIDES: Record<string, CompanyOverride> = {
  "company-id": {
    logoPosition: "left" | "top" | "contacts" | "default",
    // другие настройки...
  },
};
```

### Варианты логотипа
| Вариант | Код | Описание |
|---------|-----|----------|
| **Стандарт** | `"left"` | Слева от названия ⭐ |
| Сверху | `"top"` | По центру над названием |
| В контактах | `"contacts"` | В блоке контактов |
| Отдельно | `"default"` | Блок под шапкой |
